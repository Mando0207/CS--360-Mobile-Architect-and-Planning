# CS--360-Mobile-Architect-and-Planning

# The Weight Tracker application will help users to monitor and keep track of their weight over time and as a result help people stay motivated towards personal health goals.
# The screens and features necessary to support the apps goals are a log in screen for users to enter username and password or create new account if new to the site. A home screen that provides users with a quick snapshot of their current progress as well as their goal weight, current weight, and a quick-add button for new entries. Also, a screen with a grid that displays historical weight entries in a structured & scrollable way
